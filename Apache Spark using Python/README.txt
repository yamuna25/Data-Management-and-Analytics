############READ ME#############

To run the code, 
1. Copy code along with the input folder.
2. Replace the paths in code.py with local path.
3. Open cmd prompt, give cd <spark\bin> path.
4. Give the following command line to run the code in spark

spark-submit path/code.py Mary 1991

Note : 1991 and Mary are sample inputs for Queries 3 and 4 of Task 3.
